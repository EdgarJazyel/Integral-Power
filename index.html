Regla de la Potencia

La base de todo. Sumas 1 al exponente y divides entre ese nuevo número.

¿Qué es la integración por potencia?

La integración por potencia es uno de los métodos más básicos del cálculo integral. Se utiliza para integrar funciones que contienen potencias de la variable, como x, x², x⁵, etc.

Este método consiste en sumar 1 al exponente y dividir entre el nuevo exponente.

Fórmula de la regla de la potencia

Para cualquier exponente n, excepto cuando n = -1:

∫ xⁿ dx = xⁿ⁺¹ / n + 1 + C

Donde:

  • n = exponente de la variable.
  • C = constante de integración.

⚠️ Importante: Esta regla no se aplica cuando n = -1.

En ese caso: ∫ 1/x dx = ln|x| + C

Pasos para resolver una integral por potencia

  1. Identificar el exponente de la variable.
  2. Sumar 1 al exponente.
  3. Dividir entre el nuevo exponente.
  4. Agregar la constante de integración C.

Ejemplo Detallado

Resolver: ∫ 5x² dx

  • Paso 1: El exponente es 2.
  • Paso 2: Sumamos 1.
    2 + 1 = 3
  • Paso 3: Dividimos entre el nuevo exponente.
    5x³ / 3
  • Paso 4: Agregamos la constante.
    (5x³ / 3) + C

📌 ¿Cuándo se utiliza?

La integración por potencia se utiliza cuando la función es una potencia de la variable, como: x, x², x⁵, 7x³, 10x⁸. Es el método más sencillo para resolver integrales de polinomios y suele ser el primero que se aprende en cálculo integral.

Integración por Sustitución

Ideal cuando una función aparece multiplicada por su derivada (Cambio de variable).

¿Qué es la integración por sustitución?

La integración por sustitución es un método que se utiliza para resolver integrales complicadas transformándolas en otras más simples. Consiste en reemplazar una expresión por una nueva variable, generalmente u, para facilitar el cálculo de la integral.

Este método es útil cuando dentro de la integral aparece una función y también aparece su derivada o un múltiplo de ella.

Fórmula de la integración por sustitución

Se realiza el cambio de variable:

u = f(x)

Luego se deriva:

du = f'(x) dx

Finalmente, la integral se transforma en términos de u y se resuelve más fácilmente.

Pasos para resolver una integral por sustitución

  1. Identificar la función interna de la integral.
  2. Llamarla u.
  3. Derivar esa función para obtener du.
  4. Sustituir la integral en términos de u.
  5. Integrar.
  6. Regresar a la variable original reemplazando u por la expresión inicial.

Ejemplo Detallado

Resolver: ∫ (3x² + 2)(x³ + 2x − 1)⁵ dx

  • Paso 1: Elegimos la función interna.
    u = x³ + 2x − 1
  • Paso 2: Derivamos.
    du = (3x² + 2) dx
  • Paso 3: Sustituimos.
    ∫ u⁵ du
  • Paso 4: Integramos.
    u⁶ / 6 + C
  • Paso 5: Regresamos a la variable original.
    (x³ + 2x − 1)⁶ / 6 + C

📌 ¿Cuándo se utiliza?

La integración por sustitución se utiliza cuando una integral contiene una función compuesta y también aparece su derivada (o un múltiplo de ella), lo que permite hacer un cambio de variable para simplificar la integración.

Integración Por Partes

Fórmula salvavidas: ∫ u dv = uv − ∫ v du

¿Qué es la integración por partes?

La integración por partes es un método para resolver integrales que contienen el producto de dos funciones. Consiste en integrar una función y derivar la otra para simplificar la integral.

Fórmula de la integración por partes

∫ u dv = uv − ∫ v du

Donde:

  • u = función que se deriva.
  • du = derivada de u.
  • dv = función que se integra.
  • v = integral de dv.

Esta fórmula transforma una integral difícil en otra más sencilla.

¿Qué es la regla ILATE?

La regla ILATE es una guía para decidir qué función elegir como u cuando se aplica la integración por partes.

El orden de prioridad es:

  • 🔹 I → Funciones Inversas (arcsen, arctan, etc.)
  • 🔹 L → Funciones Logarítmicas (ln x)
  • 🔹 A → Funciones Algebraicas (x, x², x³...)
  • 🔹 T → Funciones Trigonométricas (sen x, cos x, tan x)
  • 🔹 E → Funciones Exponenciales (eˣ, 2ˣ...)

La función que aparezca primero en la lista normalmente se elige como u.

Ejemplo Detallado

Resolver: ∫ x eˣ dx

  1. Elegimos:
    • u = x (algebraica)
    • dv = eˣ dx (exponencial)
  2. Derivamos e integramos:
    • du = dx
    • v = eˣ
  3. Aplicamos la fórmula:
    ∫ x eˣ dx = x eˣ − ∫ eˣ dx
  4. Integramos:
    x eˣ − eˣ + C
