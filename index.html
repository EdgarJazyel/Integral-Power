¿Qué es una integral?

Una integral es una operación matemática que permite calcular el área bajo una curva o encontrar la función original a partir de su derivada. Se representa con el símbolo ∫ y es una herramienta fundamental para resolver problemas relacionados con áreas, volúmenes, desplazamientos y acumulación de cantidades.
